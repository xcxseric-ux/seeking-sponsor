# 求包养技能

> 「先听懂对方没说出口的话，再把自己的价值讲清楚。」

帮用户把自己的能力、价值、独特性打包成一份有吸引力的「求包养宣言」，用于寻找赞助者、金主、合作伙伴或长期客户。

## 核心理念

求包养 ≠ 乞讨，也 ≠ 操控。本质是 **价值交换 + 真实连接**：
- **你有价值** → 但别人不知道 → 学会展示
- **对方有需求** → 但她自己也未必说得清 → 学会看见
- **匹配是双向的** → 不是你追她，也不是她追你 → 是你们都看清了再选择

本技能帮你完成：**理解人性 → 自我盘点 → 价值包装 → 渠道匹配 → 话术输出 → 关系维护**

## Phase 0: 人性地基（双模式共享前置）

> 在做任何包装、改造、话术之前，先理解一件事：**所有「求被人选中」的本质，是把对方的真实需求看清楚。**
>
> 所有话术和招式都建立在这一层之上。地基不稳，招式就是 PUA。

### 0.1 心理学三件套

#### A. 马斯洛需求层次 — 你的目标卡在哪一层？

```
[自我实现]   ← 富裕金主 / 富婆主要在这里
[尊重]       ← 同上，对「被重视、被理解」高度敏感
[社交/归属]  ← 物质充足后反而更稀缺
[安全]       ← 已满足
[生理]       ← 已满足
```

**关键洞察：** 普通人争的是下三层（房贷、稳定、社交圈），富裕者争的是上两层。
- 你提供他们已经有的东西（钱、稳定、人脉）= 没价值
- 你提供他们稀缺的东西（**被理解、被尊重、被需要的感觉、自我实现的协助**）= 高价值

> **大白话：** 富婆不缺男人，缺的是「在她面前不演的那个男人」。
> 老板不缺人才，缺的是「不用他操心还能补他短板的那个搭档」。

#### B. 依恋理论（Bowlby / Ainsworth）— 三种模式 × 三种应对

成年人的关系模式来自婴幼儿期的依恋经验，主要分三类：

| 类型 | 行为特征 | 你应对的关键 | 招式适配提醒 |
|------|----------|--------------|---------------|
| **安全型** | 信任、边界清晰、不怕分离、能直接表达需求 | 平等相处，做自己 | 全部适用，长期合作最优解 |
| **焦虑型** | 怕被抛弃、需要持续确认、容易过度解读细节 | **一致性 > 浪漫**；绝不忽冷忽热 | ⚠️ 猫绳理论 = 灾难 |
| **回避型** | 不愿暴露、独立强、抗拒亲密、需要个人空间 | 给空间，不追问；让她主动靠近 | ✅ 猫绳理论有效 |

**应用：** 「保持距离感、制造神秘感」对回避型加分，对焦虑型反而是伤害。**先识别类型，再用招式。**

> **大白话：** 同一句「我最近有点忙」，对焦虑型 = 信任崩塌，对回避型 = 加分项。看人下菜。

**怎么快速识别？**
- 第一次见面她主动表达感受、问你问题 → 偏安全型
- 频繁查岗、回复秒读秒回但解读多 → 偏焦虑型
- 主动联系少、回复延迟、聊天保持距离 → 偏回避型

#### C. 自我决定理论 SDT（Deci & Ryan）— 三大底层心理需求

任何人长期投入一段关系，必须满足这三件事：

| 需求 | 含义 | 给富婆 / 金主的具体应用 |
|------|------|--------------------------|
| **自主感 (Autonomy)** | 我能选、我说了算 | 不替她决定，给 2-3 个选项让她选 |
| **胜任感 (Competence)** | 我有能力、我擅长 | 真诚让她展示她擅长的领域，请教而非炫耀 |
| **关联感 (Relatedness)** | 我和某人有意义的连接 | 制造「我们」感，共享小秘密 |

**关键反例：** 你送她再贵的礼物，如果损害她的自主感（替她安排行程、替她决定餐厅）= 负分。

> **大白话：** 高级关系的本质是「让对方在你这里更像她自己」，而不是「让她变成你想要的样子」。

### 0.2 哲学三件套

#### A. 叔本华「意志与表象」— 富人为何陷入虚无？

> 叔本华认为，人被欲望驱动：欲望未满足时是痛苦，满足后是无聊。
> 唯一的出路是**审美的瞬间**或**被深刻理解的瞬间**——能让人短暂超越欲望循环。

**应用到富裕者：**
- 富婆什么都买得起 → 持续陷在「满足后无聊」的循环里
- 你能提供的稀缺品 = **让她暂时跳出这个循环的体验**（深度对话、审美陪伴、安静的存在感）
- 不是给她更多东西，是给她**少而纯粹的瞬间**

> **大白话：** 她想要的不是第 11 个包，是「第一次有人陪她安静看完一场雨」。

#### B. 马丁·布伯「我-你 vs 我-它」关系 — 反 PUA 的哲学根基

> 出自《我与你》(Ich und Du, 1923)。布伯把所有人际关系分为两种心态：
> - **「我-它」(I-It)** ：把对方当工具、资源、达成目标的手段
> - **「我-你」(I-Thou)** ：把对方当一个独立完整的存在去相遇

**关键洞察：**
- 同一个招式（送礼、陪伴、共情），用「我-它」心态做 = PUA / 舔狗 / 骗炮，对方迟早能感觉到，关系必然崩
- 用「我-你」心态做 = 真正的吸引，对方想留下

**这是本技能最根本的一条。** 后面所有招式都建立在这一条上。

> **大白话：** 你心里把她当 ATM，她迟早能感觉到。把她当一个完整的人，她反而想留下。
> 别人都把她当「资源」，你不当 = 你的最大稀缺性。

#### C. 庄子「无用之用」— 不功利的陪伴最珍贵

> 《庄子·人间世》：「人皆知有用之用，而莫知无用之用也。」

**应用：**
- 富婆身边全是有所求的人（员工、合作方、亲戚、献殷勤的人）
- 你能提供的稀缺品 = **真的不为了什么的陪伴**
- 关键词：散步、看电影、安静坐着、不聊正事

⚠️ **这不是装出来的招式。** 装的会被识破。这是一种心态——你确实有自己的事业和生活，不需要从她身上拿什么，所以陪她就是纯陪。

> **大白话：** 全世界都在算计她，你不算计 = 你赢。但「不算计」必须是真的，靠装迟早穿帮。

### 0.3 综合产出：富裕者真实需求画像（按稀缺度排序）

把三件套拼起来，富婆/金主的真实需求清单：

| 排名 | 真实需求 | 对应理论 | 你能怎么提供 |
|------|----------|----------|--------------|
| 🥇 | **被理解、被看见**（不带评价） | 马斯洛-尊重 + SDT-关联 + 布伯-我你 | 共情倾听（见 L2.4） |
| 🥈 | **稳定可预期的存在**（不焦虑） | 依恋理论-安全型塑造 | 一致性行为（见 L2.5） |
| 🥉 | **超越日常的瞬间**（短暂出离） | 叔本华-审美超越 | 深度对话 / 共同审美体验 |
| 4 | **真诚的不算计** | 庄子-无用之用 | 真的不索取的陪伴 |
| 5 | **自主感** | SDT-自主 | 给选项不替她决定 |
| 6 | **胜任感** | SDT-胜任 | 真诚捧她擅长的领域 |
| 7 | **新鲜感、神秘感** | （表层心理） | 基础招式仍然有效 |

> **关键观察：** 只做表层情绪价值不够，前 6 名才是关系能长期稳定的深层条件。
> 表层招式 + 深层理解 = 不会崩的关系。

## Phase 1: 自我盘点（你是谁、能干啥）

### 1.0 预填充（先挖后问）

**不要上来就问用户。** 先用 `session_search` 搜索用户历史会话，挖掘已有信息：
- 搜索关键词：`项目 OR 开发 OR 部署 OR 业务 OR 客户 OR 经历`
- 从历史中提取：技术栈、项目经历、部署环境、业务领域、性格偏好
- 只对缺失信息向用户提问，已知的直接填充

这样用户只需要补充 1-2 个盲点，而不是从头回答所有问题。

### 1.1 能力资产清单

基于预填充 + 对话式挖掘，梳理以下维度：

| 维度 | 挖掘方向 | 输出 |
|------|----------|------|
| **硬技能** | 编程/设计/写作/运营/销售/管理 — 你能交付什么？ | 技能标签 + 能力等级 |
| **软技能** | 沟通/协调/执行力/抗压/学习能力 — 你的隐性价值？ | 3-5 个核心特质 |
| **资源禀赋** | 人脉/渠道/数据/设备/时间 — 你有什么别人没有的？ | 独特资源列表 |
| **过往战绩** | 做成过什么？数字说话 — 收入/用户量/效率提升 | 3 个可量化的成果 |
| **性格底色** | 你是哪种人？实干/创意/靠谱/有趣/抗压 | 人设关键词 |

### 1.2 反向挖掘（你不想做什么）

- ❌ 绝不接受的底线（道德/法律/时间/地点）
- ❌ 不擅长的领域（别吹自己没有的）
- ❌ 不匹配的关系类型（全职/兼职/项目制/长期陪伴）

### 1.3 自我依恋类型识别

用 Phase 0.1.B 同样的框架问自己：你自己是什么依恋类型？
- 你是焦虑型 + 找回避型金主 = 互相伤害（你越追她越退）
- 你是回避型 + 找焦虑型金主 = 她崩溃你逃跑
- **匹配度自检比包装更重要。**

## Phase 2: 价值包装（怎么把自己卖出去）

### 2.1 定位策略 — 三选一

根据 Phase 1 的盘点结果，选择最适合的定位：

| 策略 | 适用场景 | 核心话术 |
|------|----------|----------|
| **「铁锹型」** | 你有明确可交付的技能 | 「你出钱，我出活，按结果说话」 |
| **「种子型」** | 你有潜力但没战绩 | 「便宜的时候买我，贵了就买不起了」 |
| **「拼图型」** | 你能补对方缺的那块 | 「你缺的不是员工，是能补刀的人」 |

### 2.2 价值主张公式

```
我是 [一句话定位]
我能帮你 [解决什么问题/创造什么价值]
我的方式是 [怎么干活]
你只需要 [对方要付出什么]
结果你得到 [具体收益]
```

**示例：**
> 我是会写代码的运营。我能帮你把业务流程自动化、省掉 2 个岗位的人力成本。我的方式是深入业务场景再写代码，不是闭门造车。你只需要每月固定合作费，结果你得到一个能当技术用的运营、能当运营用的技术。

### 2.3 差异化锚点

在价值主张基础上，找 1-2 个让人记住的标签：
- 「那个会 XX 的 XX」
- 「XX 行业里最懂 YY 的人」
- 「前 XX 大厂，现自由职业」
- 不是吹牛，是找到你的 **唯一性**

### 2.4 Cialdini 影响力六原则在求包养中的应用

> 出自 Robert Cialdini《影响力》。这不是操控技巧，是**让对方更容易看到你价值**的认知规律。

| 原则 | 含义 | 应用 |
|------|------|------|
| **互惠 (Reciprocity)** | 收到好处会想回报 | 第一次见面带个有用的小东西 / 主动帮一个小忙，**不索要回报** |
| **承诺一致 (Commitment)** | 做过承诺会倾向坚持 | 让对方在小事上先答应你 → 大事更易合作 |
| **社会认同 (Social Proof)** | 别人选了你 = 你靠谱 | 案例 / 推荐人 / 成功故事自然提及，不要堆砌 |
| **喜爱 (Liking)** | 喜欢的人更容易合作 | 真诚 + 共同点 + 适度赞美（具体而非空洞） |
| **权威 (Authority)** | 专家说的更可信 | 在你的领域里有可证明的成果 / 标签 |
| **稀缺 (Scarcity)** | 稀缺的更想要 | 不要随时有空。你是有自己事业的人。 |

⚠️ **边界：** 这六条用来「让对方更容易识别你的真实价值」是健康的；用来「让对方在没价值的情况下也选你」是操控，迟早翻车。

## Phase 3: 渠道匹配（去哪找金主）

### 3.1 金主画像

| 金主类型 | 特征 | 在哪找到 | 最佳切入方式 |
|----------|------|----------|-------------|
| **创业老板** | 缺人、缺时间、缺靠谱搭档 | 创业社群、行业活动、朋友圈 | 帮他解决一个具体问题（免费小活） |
| **内容创作者** | 缺技术支持、缺运营、缺流量 | B站/抖音/小红书/X 的中腰部创作者 | 「我看了你的内容，我能帮你做 XX」 |
| **投资机构** | 投后服务、投后管理、需要技术尽调 | 行业活动、FA、LinkedIn/脉脉 | 证明你对某个领域有深度理解 |
| **企业部门** | 外包需求、灵活用工、项目制合作 | 猎头、BOSS直聘、行业社群 | 展示过往案例 + 快速响应 |
| **个人/小团队** | 需要「一个人顶一个团队」 | 独立开发者社群、自由职业平台 | 性价比 + 靠谱度 |

### 3.2 渠道优先级

1. **熟人网络**（转化率最高）→ 朋友圈/同事/前老板
2. **垂直社群**（匹配度最高）→ 行业群/技术社区
3. **公开平台**（覆盖面最广）→ 飞书/即刻/小红书/微博
4. **被动吸引**（长期积累）→ 开源项目/技术博客/个人网站

### 3.3 金主信任曲线（基于 Phase 0 心理学）

不要妄图一次到位。信任建立有自然节奏：

```
第一次接触     →  你不是骗子（基础门槛）
第 2-3 次       →  你确实懂行（胜任感建立）
第 4-6 次       →  你说话算话（一致性，安全型依恋触发）
第 7-10 次      →  你不只是为了赚钱（庄子-无用之用，差异化）
长期            →  你是「我的人」（关联感，SDT 满足）
```

每一步对应 Phase 0 的某个理论。**跳步 = 翻车**。

## Phase 4: 话术输出（怎么开口说）

### 4.1 话术模板库

根据场景选择：

**A. 朋友圈/社交平台版本（轻松型）：**
```
最近在找新的合作机会。
简单说：[你的核心能力]，[能解决什么问题]。
如果你或者你认识的人正好需要，聊聊？
不是求职，是找个能互相成就的搭档。
```

**B. 私聊/精准触达版本（正式型）：**
```
[称呼]，冒昧打扰。

我看到 [你/你们] 最近在 [对方的事情/项目]，
我刚好在 [相关领域] 有些经验——
[一句话你的价值]。

如果感兴趣，可以约个时间聊聊？
不合适也没关系，就当认识一下。
```

**C. 个人介绍/简历版本（结构型）：**
```
[一句话标签]

我能做什么：
• [能力1]：[具体说明 + 成果]
• [能力2]：[具体说明 + 成果]
• [能力3]：[具体说明 + 成果]

我想要什么：
[期望的合作方式 + 对方能获得什么]

联系方式：[渠道]
```

**D. 搞笑/破冰版本（整活型）：**
```
本人 [标签]，长期寻求包养。

包养方案如下：
• 月卡：[价格/条件] — [包含什么]
• 季卡：[价格/条件] — [包含什么 + 额外]
• 年卡：[价格/条件] — [包含什么 + 更多额外]

支持定制。不支持白嫖。
```

### 4.2 跟进节奏

| 阶段 | 时间 | 行动 | 话术 |
|------|------|------|------|
| **触达** | Day 0 | 发出第一条消息 | 见上模板 |
| **轻跟进** | Day 3 | 如果没回，补充一条有价值的信息 | 「对了，我刚好做了个 XX，你看看有没有参考价值」 |
| **深度跟进** | Day 7 | 如果有兴趣但没推进，约具体时间 | 「要不约个 15 分钟电话？我准备个简单的方案你看看」 |

# Mode B: 躺平模式 — 被富婆包养修炼体系

> 「年少不知富婆好，错把少女当成宝。年少不知软饭香，错把青春倒插秧。」

当用户触发词包含「躺平/被富婆包养/不想努力了/软饭硬吃」时，进入此模式。

本模式用心理学 + 哲学地基约束具体招式，目标不是套路，而是真懂人。

### 核心心法

> **外表是门票，灵魂是续费。但灵魂不是装出来的，是你真的看懂了她。**

**真实理解 > 情绪价值 > 皮囊 > 金钱。**

富婆的心理画像（基于 Phase 0 综合）：
- **马斯洛层面**：物质满足，卡在尊重 + 自我实现层
- **依恋层面**：高净值人群焦虑型 + 回避型比例显著高于普通人群（长期承担压力 / 信任成本高）
- **SDT 层面**：自主感强（习惯掌控），但关联感和真实关系稀缺
- **叔本华层面**：欲望循环饱和，急需「短暂超越」的体验
- **布伯层面**：身边全是「我-它」关系（员工、合作方、献殷勤的人），「我-你」关系几乎不存在
- **庄子层面**：身边没有不算计她的人

> **所以她真正想要的，不是更多礼物，是一个能让她「短暂当个普通人」的存在。**

### L2.2 适当距离感 ≠ 间歇性强化

> **必须澄清边界：**
> - **健康做法：保持你自己的生活节奏 → 自然产生距离感**
> - **PUA 做法：故意忽冷忽热制造焦虑 → 操控**
>
> 区别在动机：你是真的有自己的事，还是装出来的？

✅ **健康的距离感：**
- 你确实在忙自己的事业，不是 24h 待命
- 回复有自然节奏，不刻意快也不刻意慢
- 有自己的朋友、爱好、独处时间
- 这不是策略，这是你本来的样子

❌ **PUA 式的间歇性强化（绝对避免）：**
- 故意秒回再消失，制造焦虑
- 用冷淡惩罚、用热情奖励，训练她的反应
- 让她处于「不知道你下一步会怎样」的不安中
- 利用她的焦虑型依恋特征做情绪剥削

> **检验标准：** 如果你的招式在对方完全清楚的情况下还能成立，是健康的；
> 如果一旦对方知道你在「策略性操作」就会崩溃，那就是 PUA。

### L2.3 态度三要素

1. **男子气概** — 有主见、有担当、不优柔寡断
2. **迷之自信** — 「大丈夫生居天地之间，岂能郁郁久居人下！」
3. **持续学习** — 知识储备是幽默和谈吐的根基

> **补充：** 自信的根基是「我有自己的事业 + 我看得懂人」。
> 没事业的自信 = 嘴硬，看不懂人的自信 = 油腻。

### L2.4 共情倾听 4 步法（Carl Rogers 人本主义心理学）

> 这是 Phase 0.3 第 1 名「被看见」的具体操作方法。**最稀缺、最难、最有效。**

#### 4 步骤

```
Step 1: 识别情绪 — 她说话时背后的情绪是什么？（生气？委屈？疲惫？兴奋？）
Step 2: 命名情绪 — 用一个词把它说出来：「你听起来挺累的」
Step 3: 不评判反映 — 不要说「我懂」「我也是」「你应该……」，而是反映：「这个项目压了你这么久，确实让人喘不过气」
Step 4: 等待回应 — 沉默 3 秒，让她接话
```

#### 三个常见场景脚本

**场景 A：她抱怨工作上的人**
```
她：「我那个合伙人最近真的让我心累，什么都要我决定。」

❌ 错误回应：
- 「那你换个合伙人吧」（替她决定，损害自主感）
- 「我也遇到过这种人」（抢话，转向自己）
- 「他怎么这样」（廉价共鸣）

✅ 正确回应：
- 「听起来你最近一个人扛了很多事。」（命名情绪 + 反映）
- 然后停顿，等她接话
```

**场景 B：她说自己最近压力大**
```
她：「最近真的好累，什么都不想做。」

❌ 错误回应：
- 「那你休息下啊」（建议）
- 「我帮你做」（抢救援者角色）
- 「你都这么有钱了还累什么」（最致命，否定她的感受）

✅ 正确回应：
- 「能感觉到你最近是真的撑得很满。」（共情）
- 「这种时候不需要做什么，是吗？」（开放式问题）
- 然后等她接话
```

**场景 C：她兴奋地分享一件事**
```
她：「我今天谈成那个客户了！」

❌ 错误回应：
- 「厉害」（廉价）
- 「我就知道你能行」（家长式）
- 「下一单是哪个？」（转向工具感）

✅ 正确回应：
- 「这个客户你是不是已经盯了快半年了？」（被看见 = 你记得她说过的细节）
- 「能想象你今天回家路上有多爽。」（共情她的情绪而非事件）
```

#### 共情 vs 同情（Brené Brown）

| 同情 (Sympathy) | 共情 (Empathy) |
|-----------------|----------------|
| 「真可怜」 | 「我能感受到你在那个状态里」 |
| 站在井外看井里的人 | 下到井里陪她一起 |
| 制造距离 | 制造连接 |

> **大白话：** 大多数人对富人都是「同情」（怜悯她也有烦恼）或「羡慕」（你都那样还有什么烦恼）。
> 共情是稀缺品。你能做这个，你就是她的稀缺品。

### L2.5 安全型依恋的塑造（针对焦虑型 / 回避型富婆）

> 高净值人群焦虑型/回避型比例较高（长期承压 + 信任成本高）。
> 「让她在你这里逐渐变成安全型」= 你成为她生命里独特的那个人。

#### 三大行为锚点

**1. 一致性 (Consistency)** — 焦虑型的解药
- 你说几点到就几点到
- 你说做的事就做完
- 你的情绪稳定，不被她的情绪带走
- 不忽冷忽热

**2. 可预期 (Predictability)** — 不让她猜
- 不主动联系不代表冷落 → 提前告知「我下周在出差，可能少回消息」
- 不要让她处于「他怎么了？是不是讨厌我？」的状态
- 焦虑型最怕未知

**3. 不情绪化反应 (Non-Reactivity)** — 回避型的解药
- 她试探你时（突然冷淡、突然挑衅）→ 你不慌不躁
- 她道歉时 → 你不抓住小辫子放大
- 她偶尔疏远时 → 给空间，不追问
- 回避型最怕「我一犯错就被审判」

> **三条都做到 = 你成为她在这个世界上罕见的「安全基地」。** 这个位置一旦确立，几乎不可替代。

# 操控 vs 吸引：红线专章

> 所有 Mode B 招式如果跨过这条线，就是 PUA，迟早翻车，并对对方造成实际伤害。

## 核心区别

| 维度 | ✅ 吸引 (Attraction) | ❌ 操控 (Manipulation / PUA) |
|------|--------------------|------------------------------|
| **心态** | 我-你（布伯）：把对方当独立完整的人 | 我-它：把对方当达成目标的工具 |
| **信息透明度** | 招式被看穿了仍然成立 | 一旦被看穿就崩塌 |
| **对方代价** | 对方变得更像自己、更幸福 | 对方失去自我、产生依赖、被消耗 |
| **可持续性** | 越久越深 | 必然在某个时间点暴露 |
| **你的代价** | 你也在成长 | 你需要持续维护谎言 |

## 高风险招式 × 边界标注

| 招式 | 健康用法（吸引） | 越界用法（PUA） |
|------|------------------|------------------|
| **保持距离感** | 你确实有自己的事 | 故意忽冷忽热制造焦虑 |
| **神秘感** | 不一次亮所有底牌（自然） | 编造身份/经历，制造虚假悬念 |
| **制造惊喜** | 真心实意的小心思 | 用奖励/撤回训练她的反应 |
| **共情倾听** | 真的想理解她 | 套出她的弱点为后续操控用 |
| **真诚夸赞** | 你真的觉得她那点好 | 廉价讨好，话术化 |
| **「让她主动」** | 给她自主感 | 通过冷暴力逼她追你 |
| **示弱激发保护欲** | 真实的脆弱时刻分享 | 装可怜博取同情 / 经济援助 |

## 自检三问

每次用任何招式之前，问自己：

1. **如果她完全知道我此刻在做什么、为什么这么做，这个招式还能成立吗？**
   - 能 → 是吸引
   - 不能 → 是操控

2. **她在这段关系里是变得更像自己、更舒展，还是更不像自己、更焦虑？**
   - 更像自己 → 是吸引
   - 更焦虑 → 是操控

3. **如果有一天她说「我们结束吧」，我能体面退出吗？**
   - 能 → 关系是健康的
   - 不能（要纠缠、报复、伤害） → 你从一开始就走偏了

## 法律 + 伦理红线（绝对不可触碰）

- ❌ 已婚 / 有家庭的对象（民事/道德双重风险，白马会所即前车之鉴）
- ❌ 隐瞒重要信息（婚姻状况、感染病、债务）骗取财物
- ❌ 录音录像作为筹码 / 威胁
- ❌ 利用对方信任做投资骗局 / 资产转移
- ❌ 任何形式的精神虐待（煤气灯效应 / 持续否定 / 隔离她和其他社交关系）
